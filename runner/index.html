This would be a Lynx app, but no app to load was specified via the URL.

You can load an app via one of these three methods:

  1. In the editor, paste your app's code and press Open Full Window. This will put a base64-encoded and compressed version of your code into the URL hash, which you can then share with others or bookmark.
  2. In the editor, press Save and enter a name. This will store the code in the browser's local storage. Recall it here
    and then copy the URL once it loads. Note that this method only works on this device and this browser.
  3. Save your app's code somewhere else (such as Github Gist), then find the "raw" URL provided by the service. (For a Gist, the "raw" URL looks like https://gist.githubusercontent.com/<your username>/<gist ID>/raw/<revision SHA>/<filename>.) Then load it here
    and copy the URL once it loads.
